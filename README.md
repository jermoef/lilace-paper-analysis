# lilace-paper-analysis
Code for [Lilace paper](https://www.biorxiv.org/content/10.1101/2025.06.24.661380v1.full) analyses

## Directory Structure
```
config/   - contains snakemake config files
figures/  - contains Rmarkdown files for paper figures
src/      - contains model and analysis code
workflow/ - contains snakemake pipeline for simulations and data analysis
```
